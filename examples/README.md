# Examples

倉庫內 Vite 範例。`@cluion/*` 經 alias 指向 `packages/*/src`。

| 目錄 | 入口 | 指令 |
|---|---|---|
| `html/` | `createApp` | `pnpm example:html` |
| `vue/` | `createApp` | `pnpm example:vue` |
| `react/` | `createApp` | `pnpm example:react` |
| `laravel/` | ProjectStore 參考 | 見目錄 README |

```bash
pnpm install
pnpm example:html
pnpm dev
```
